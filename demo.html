Instructions

These demos use the Web Audio and MIDI API which seems to be available only in Chrome. In order to use the MIDI you need to enable it in Chrome by typing the following address chrome://flags/#enable-web-midi. You can learm more about the Web Audio and MIDI API Here.

Connect you MIDI keyboard/controller and open this page. By default, this page opens all the available MIDI inputs. Load one of the demos and click 'Update' to run it. Modify the function controlChange to respond to the MIDI messages you want. Modify the code and click 'Update' to recompile it. All the demos have a volume parameter controlled by the control 30.
