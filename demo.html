Vult generates Javascript that can be directly run in you browser and process or synthesize audio. This platform was created during the MIDIHACK 2015 in Berlin. The project Vult leaves the console

Requirements

These demos use the Web Audio and MIDI API. In order to use the MIDI you need to enable it in Chrome by typing the following address chrome://flags/#enable-web-midi. You can learm more about the Web Audio and MIDI API Here.

For some of the demos you will need to have a MIDI keyboard/controller connected. By default this page opens all the availables MIDI ports. You may have problems if other software is using the MIDI ports.
